## Installation

Install my-project with npm

```bash
cd portfolio
npm i   
```
    
## Deployment
To config the url of the page , you need to modify "base" in vite.config.js
All comands are config in package.json

To run locally :
```bash
  npm run dev
```
To deploy to gh-pages: 
```bash
  npm run deploy
```
